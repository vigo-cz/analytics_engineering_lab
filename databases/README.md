# Database Experiments

Experiments with various database technologies.

## Databases

- **DuckDB**: Embedded analytical database for OLAP workloads
- **ClickHouse**: Column-oriented OLAP database
- **PostgreSQL**: General-purpose relational database
- **TimescaleDB**: Time-series extension for PostgreSQL

## Structure

Each database has its own subdirectory with:
- Schema definitions
- Sample queries
- Performance benchmarks
- Migration scripts
